## 设计经验

- 让 AI coding 提效的方法：
  1. 先写 spec（架构、接口、数据流），让 AI 评审后再实现
  2. 使用 mypy/pyright 静态检查，提前捕获字段名错误
  3. 测试驱动，先写测试再实现
  4. 把设计决策写入 MEMORY.md，每次修改前让 AI 读取
  5. 复杂逻辑拆小，先实现最小可运行版本
  6. 让 AI 做 code review，提前发现问题
